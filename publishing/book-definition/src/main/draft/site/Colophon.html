A number of technologies were used in the production of this book:

  • The code is, of course, primarily written using the Groovy distribution
  • The book is assembled using a small Gradle-based Groovy application I strung together.
    • You can find the source here
  • MultiMarkdown Composer was used as my primary editor
  • The text has been converted using the MultiMarkdown tool.
  • The website uses the following elements:
    • Syntax highlighting by highlight.js
    • Equations styled by MathJax
    • Page layout is based on the Perfect ‘Left Menu’ 2 Column Liquid Layout by Matthew James Taylor
  • The Groovy/Grails Tool Suite was used to test some of the sample code and write the book assembler.

I wish I had a 19th century engraving on the cover so that I could tell you about a really cool animal. If I did I would use the Pied butcherbird, perhaps accompanied with a picture from one of John Gould’s books. I would then tell you that this small-ish bird has a beautiful song and a really friendly composure. My resident (wild) butcherbirds like to sing on my deck when it’s raining, follow me when I mow and, occasionally, bathe under the sprinkler on hot days.
